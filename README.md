# NemOS

> **Recovered Archive Build v0.13**
>
> *"Some files may be corrupted. Some files may be missing. Some files may be lying."*

## About

NemOS is a browser based operating system experience created for Hack Club's Stardance program.

At first glance, it appears to be a simple retro desktop environment featuring draggable windows, file explorer, notes, and a boot sequence.

However, hidden beneath the desktop is a fragmented archive documenting the life of an individual known only as **Nemo** and a mysterious project called **NULL**.

As more files are recovered, the truth behind the archive slowly reveals itself.

---

## Current Features

### Desktop Environment

* Draggable windows
* Window focus system (active window comes to front)
* Resizable application windows
* Desktop icons with single-click selection
* Double-click application launching
* Custom retro UI

### Boot System

* Animated boot sequence
* Archive recovery messages
* Corruption warnings
* Interactive Y/N access prompt
* Lore integrated startup experience

### Applications

#### README_FIRST.txt

The first recovered file.

Acts as both:

* A notes application
* The beginning of the archive narrative

#### ARCHIVE_EXPLORER.exe

A custom file explorer used to browse recovered records.

Currently supports:

* Folder navigation
* File unlocking
* Text document viewing
* Progressive archive recovery

### Narrative Systems

* File based storytelling
* Unlockable records
* Corrupted archive structure
* Hidden information
* Ongoing mystery surrounding Project NULL
